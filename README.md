## About

I am an **Electronics Apprentice (EFZ) in Switzerland** with a strong interest in software engineering, embedded systems and cybersecurity.

I build practical systems where software interacts with hardware, live data and technical workflows. My projects include AVR firmware, real-time Rust/Python communication, signal processing, monitoring dashboards and Linux-based deployments.
