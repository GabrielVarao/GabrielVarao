## About

I am an **Electronics Apprentice (EFZ) in Switzerland** with a strong focus on software engineering, embedded systems and cybersecurity. I build projects where code interacts with real hardware, live data and operational workflows - from AVR firmware and real-time signal pipelines to monitoring dashboards and embedded Linux deployments.

My engineering approach combines **clear architecture, practical testing, maintainable code and product-oriented interfaces**. I enjoy taking a technical problem from the first prototype through implementation, visualization, validation and deployment.

I am currently expanding my knowledge in **AI/ML, signal processing, full-stack development, secure software design and Linux-based product engineering**. My long-term goal is to work professionally in cybersecurity and build technology that is both technically strong and genuinely useful.

### Open To

- Open-source collaboration in Rust, Python, embedded systems and developer tooling
- Junior software engineering, cybersecurity and applied AI opportunities
- Projects involving real-time data, automation, monitoring or hardware integration
- Learning from experienced engineers and contributing to ambitious technical teams

## Featured Projects

<details>
<summary><strong>Rust Signal Filter Lab</strong> - Real-time cross-language signal pipeline</summary>

<br />

A real-time measurement simulation in which Rust generates a clean signal, adds approximately 5% noise, applies a moving-average filter and streams typed samples to Python for live visualization.

| Category | Engineering Detail |
|---|---|
| **Stack** | Rust, Python, ZeroMQ, Protocol Buffers, Matplotlib |
| **Scale** | Continuous inter-process stream with clean, noisy and filtered samples |
| **Performance** | Compact binary serialization and responsive live graph updates |
| **Security** | Explicit Protobuf schema, typed data contract and controlled local communication |
| **Impact** | Demonstrates distributed architecture, signal processing and real-time observability |
| **Repository** | [View Rust Signal Filter Lab](https://github.com/GabrielVarao/sinus_signal_filter) |

The project separates data generation, filtering, serialization, transport and presentation into clear components. It is designed as a practical foundation for sensor simulation, industrial monitoring and future OPC UA or GUI extensions.

</details>

<details>
<summary><strong>4x4 Matrix Keypad Calculator</strong> - Hardware-validated AVR firmware</summary>

<br />

A firmware calculator for the ATmega2560 using a 4x4 matrix keypad as input and an HD44780-compatible LCD for immediate output.

| Category | Engineering Detail |
|---|---|
| **Stack** | C, AVR, ATmega2560, Microchip Studio, HD44780 LCD |
| **Scale** | Embedded single-board application with keypad and display integration |
| **Performance** | Deterministic finite-state-machine control and immediate input feedback |
| **Security** | Explicit state transitions and protection against division by zero |
| **Impact** | Combines embedded input handling, arithmetic logic and real hardware validation |
| **Repository** | [View Keyboard Calculator](https://github.com/GabrielVarao/Keyboard-Calculator) |

The architecture supports multi-digit input, floating-point calculations, result reuse and the four standard arithmetic operations. The complete workflow was tested on physical hardware.

</details>

<details>
<summary><strong>Stepper Motor Controller</strong> - Deterministic motion-control firmware</summary>

<br />

A structured AVR firmware project that controls stepper motor direction and speed through a finite state machine and an improved half-step sequence.

| Category | Engineering Detail |
|---|---|
| **Stack** | C, AVR, ATmega2560, Microchip Studio, stepper motor driver |
| **Scale** | Physical motor-control system operating from 20 to 300 steps per second |
| **Performance** | Extended eight-state half-step sequence for smoother motion and finer positioning |
| **Security** | Predictable state transitions and bounded speed-control behavior |
| **Impact** | Demonstrates embedded control, hardware integration and engineering validation |
| **Repository** | [View Stepper Motor Controller](https://github.com/GabrielVarao/Schrittmotor) |

The implementation supports clockwise and anticlockwise rotation, runtime speed adjustment and hardware-tested behavior. Its modular FSM structure makes future modes such as pause or idle straightforward to add.

</details>

<details>
<summary><strong>Rust Sinus Console</strong> - Mathematical ASCII visualization</summary>

<br />

A compact Rust application that calculates a sine wave and renders it directly in the terminal as an ASCII graph.

| Category | Engineering Detail |
|---|---|
| **Stack** | Rust, Cargo, standard library mathematics |
| **Scale** | Configurable terminal visualization with adjustable width, height and frequency |
| **Performance** | Lightweight single-process calculation with direct console rendering |
| **Security** | No network services, external data sources or privileged operations |
| **Impact** | Clear introduction to Rust loops, numeric mapping and visualization logic |
| **Repository** | [View Rust Sinus Console](https://github.com/GabrielVarao/sinus_console) |

The project maps calculated sine values to terminal rows and adds a visual center axis. It provides a clean starting point for more advanced signal-generation and visualization projects.

</details>
